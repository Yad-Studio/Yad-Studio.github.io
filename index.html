Yad Studio - Markov algorithms IDE and interpreter

Description

Yad Studio is an easy IDE for Markov Algorithms interpretation. Read more about Markov Algorithms

Tutorial

Create firstly the Alphabeth T = {...}. Than create rules (in one row write only a one rule). Use-> and ->. This symbol ->. creates terminated rules(if the algorythm use this rule - the program will be stoped) There is an example:

T = {a,b,c,d}
aa->a
bbb->.c

Input word: bbbaaa Set it in right palen in the Input Word box. Click Run button (at the right panel). Output: ca Output will be saved in the Input history.

If you'd like to //add a comment write like this

Run with Debug

If you'd like to see the changin in the input string step by step set input word in Input Word box and click on the Run with Debug button.

Also you can put break points to the rules (click befor the line number) and than Run with Debug The program will stoped when it will be reached the first break point. Than you can use Next Step (see the next rule, which will be used) or Continue (stoped, when the algorithm reach this rule next time).

Input History

Input history saves all your previous input strings during all the sesions. If you'd like to add a line to the history, set string to Input Word box and then Run or Run with Debug the program.

Use the previous input by double click and it will be setted in the Intput Word box.

If you'd like to delete a line from the history use right click on activated row in the history and select Delete in the context menu. Also you can activate row in the Input history and click on '-' button.

Unactive Run Button

If there is a mistake in the code Run and Run with Debug will not be active. The line(or lines) with the mistake will be matched in red color. For example if you use symbols which are not from the alphabeth lines become red.

Illegal Characters

It is forbidden to use in the Alphabeth to use empty char,-,>, point, comma, T (used for Alphabeth initialisation)

Examples

Example 1:
Input: dogs_and_cats_are_friends
Output: a_dog_and_a_cat_is_enemies

//dogs_and_cats_are_friends
T = {d,o,g,s,c,a,t,r,e,i,f,n,m,_}

dogs->a_dog
cats->a_cat
are->is
friends->enemies

Example 2:
Input: AngelineJolie
Output: JamesHaven!

//convert Angeline Jolie to James Haven(her brother)
T = {A,n,g,e,l,i,J,o,a,m,s,H,v,!}

Angeline->James
Jolie->Haven!
